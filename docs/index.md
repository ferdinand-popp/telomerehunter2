# TelomereHunter2

TelomereHunter2 is a Python tool for estimating telomere content and analyzing telomeric variant repeats (TVRs) from genome sequencing data.

## Key features

- Bulk analysis from BAM/CRAM
- Tumor vs control comparison (log2 ratio)
- Single-cell BAM support (barcode-aware)
- TSV outputs plus optional HTML reports and plots
- Docker/Apptainer support

## Install

```bash
pip install telomerehunter2
```
